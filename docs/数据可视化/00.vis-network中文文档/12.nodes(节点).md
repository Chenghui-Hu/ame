处理节点的创建和删除，并包含全局节点选项和样式    

 

节点的选项必须包含在名为“nodes”的对象中。所有这些选项也可以按节点提供。显然，不应全局定义“id”，而应为每个节点定义“id”。全局节点对象中定义的选项将应用于所有节点。如果节点有自己的选项，则将使用这些选项而不是全局选项。

为节点提供选项后，将覆盖该属性的全局选项，如果该节点位于组中，还将覆盖该属性的组选项。如果您随后将该选项设置为空，它将恢复为默认值。

network.setOptions(options);
